# @rudironsoni/dotnet-harness-copilot

GitHub Copilot bundle for dotnet-harness.

## Installation

```bash
npm config set @rudironsoni:registry https://npm.pkg.github.com
npm install @rudironsoni/dotnet-harness-copilot
```

## Contents

- `bundled/.github/agents/**`
- `bundled/.github/instructions/**`
- `bundled/.github/prompts/**`
- `bundled/.github/skills/**`
- `bundled/.github/copilot-instructions.md`
